# Tugas 7 dan 8

## Ketentuan Tugas

Buatlah sebuah program Java untuk mengelola agenda kegiatan harian dengan menerapkan konsep Input/Output dan Enkapsulasi. Program harus mampu menerima data dari pengguna, menampilkan daftar agenda, serta menyimpan hasilnya ke dalam file.

### Spesifikasi Program

1. **Kelas dan Enkapsulasi**
   - Buat satu kelas khusus untuk merepresentasikan sebuah agenda kegiatan.
   - Atribut-atribut penting (misalnya waktu, deskripsi, atau nama kegiatan) harus dibuat ```private```.
   - Gunakan getter dan setter untuk mengatur akses ke atribut-atribut tersebut.
   - Sediakan sebuah method di kelas tersebut untuk menampilkan informasi agenda dengan format yang rapi.
2. **Input dan Output**
   - Program harus dapat menerima data dari pengguna.
   - Setelah data dimasukkan, tampilkan kembali daftar agenda yang sudah tersimpan ke layar.
   - Program harus dapat menyimpan seluruh agenda ke dalam file teks (.txt) dengan nama berdasarkan tanggal saat program dijalankan (misalnya 05-11-2025.txt).
   - Data agenda juga harus bisa dibaca kembali dari file tersebut ketika program dijalankan ulang. 
3. **Aturan Tambahan**
   - Minimal terdapat dua agenda yang dapat ditambahkan dalam satu kali eksekusi program.
   - Gunakan struktur data yang sesuai untuk menyimpan kumpulan objek agenda selama program berjalan.
   - Pastikan semua input stream dan output stream ditutup dengan benar setelah digunakan.

Dengan isi file sebagai berikut:
```java
// Nama : ...
// NIM  : ...

public class PPBO_07_08_NIM_NamaLengkap {
    public static void main(String[] args) {
        // ...
    }
}
```
